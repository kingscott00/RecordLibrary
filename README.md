# Scott's Vinyl Collection

A beautiful, cozy web application to browse and explore your vinyl record collection from Discogs.

## Features

- **Three-Pane Layout**: Artists (left), Albums (middle), Details (right)
- **Folder Filtering**: Filter your collection by genre/category
- **Artist Search**: Quickly find artists in your collection
- **Wikipedia Integration**: Automatic album cover images and Wikipedia links
- **Cozy Theme**: Warm, inviting design with leather and wood-inspired colors
- **iPad Air Optimized**: Responsive design perfect for tablet viewing
- **Static Site**: No server required - host on GitHub Pages

## Usage

### Viewing Locally

1. Open `index.html` in a modern web browser
2. The collection will load automatically

### Hosting on GitHub Pages

1. Push the repository to GitHub
2. Go to Settings > Pages
3. Select the branch to deploy (e.g., `main` or `claude/vinyl-collection-webapp-01YMurNXrg7cME6ED5k3LDS5`)
4. Your site will be available at `https://kingscott00.github.io/RecordLibrary/`

## Navigation

1. **Select a Folder**: Use the dropdown at the top to filter by genre
2. **Browse Artists**: Click on an artist in the left pane
3. **View Albums**: See all albums by the selected artist in the middle pane
4. **Album Details**: Click an album to view full details, cover art, and links

## Data Source

The collection data is loaded from `scottking11-collection-20251121-1936.csv`, exported from Discogs.

## Technologies

- Pure HTML, CSS, and JavaScript (no frameworks required)
- Wikipedia API for album artwork
- Responsive design with CSS Grid
- Modern ES6+ JavaScript
